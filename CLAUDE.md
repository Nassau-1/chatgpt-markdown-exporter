# Claude Repository Rules

- Work inside this repository; do not treat temporary runtime folders as the source of truth.
- Keep the repo aligned with `README.md`, `SECURITY.md`, `CHANGELOG.md`, `TODO.md`, and `docs/`.
- Use normal Markdown links in repo docs.
- Keep secrets, local `.env`, logs, and backups out of tracked files.
- Record major decisions in `docs/decisions/ADR-*.md`.
- Treat `src/` as the unpacked extension root and keep checks under `scripts/`.
